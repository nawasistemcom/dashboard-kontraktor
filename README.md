# Dashboard Kontraktor

Dashboard Kontraktor merupakan alat monitoring yang membantu developer, Project Manager, dan Site Manager mengontrol kinerja kontraktor secara objektif dan terukur.

Dengan dashboard yang baik, manajemen proyek dapat mengetahui progres pekerjaan, deviasi proyek, produktivitas tim, dan risiko keterlambatan lebih cepat.

## Fungsi Dashboard Kontraktor

- Monitoring progres pekerjaan
- Mengukur deviasi proyek
- Evaluasi kinerja kontraktor
- Mengontrol target pembangunan
- Membantu pengambilan keputusan

## Data yang Dapat Dimonitor

### Progress Fisik

Perbandingan antara target dan realisasi pekerjaan di lapangan.

### Deviasi Progress

Selisih antara progres rencana dan progres aktual.

### Produktivitas Tim

Evaluasi pencapaian pekerjaan berdasarkan target yang ditetapkan.

### Status Pekerjaan

- On Schedule
- Minus
- Plus
- Selesai

### Evaluasi Kontraktor

Monitoring performa kontraktor berdasarkan progres dan kualitas pekerjaan.

## Manfaat Dashboard Kontraktor

- Mengurangi keterlambatan proyek
- Mempermudah monitoring mingguan
- Membantu action plan proyek
- Meningkatkan kontrol pekerjaan
- Mendukung sistem kerja modern
